# 495. Implement Stack



#### Description

Implement a stack. You can use any data structure inside a stack except stack itself to implement it.Have you met this question in a real interview?  YesProblem Correction

#### Example

```cpp
push(1)
pop()
push(2)
top()  // return 2
pop()
isEmpty() // return true
push(3)
isEmpty() // return false


stack can be implemented in different data strucure, 
vector
queue
linked list


class Stack {
    
    private:
    std::vector<int> data;
    
public:

    /*
     * @param x: An integer
     * @return: nothing
     */
    void push(int x) {
        data.push_back(x);
    }
    
    /*
     * @return: nothing
     */
    void pop() {
        data.pop_back(); //vector has the 'pop_back' function
    }

    /*
     * @return: An integer
     */
    int top() {
        //assert(data.size() > 0);
        return data[data.size() -1 ];
    }
    
    /*
     * @return: True if the stack is empty
     */
    bool isEmpty() {
        return data.size() == 0;
    }
};
```
